# Contributing

Contributions should preserve Work Wiki's default privacy posture.

## Rules

- Use `.example.md` for public source registry samples.
- Use aliases instead of real local paths, account names, tenant names, ticket keys, or private provider identifiers.
- Do not add raw email, chat, document, monitoring, incident, or code evidence to examples.
- Keep protocols Markdown-first and agent-portable.
- Update validation guidance when adding new source types or templates.
