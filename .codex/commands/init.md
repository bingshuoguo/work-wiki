# /init

Initialize this Work Wiki workspace.

Use `.codex/skills/init-workwiki/SKILL.md` and follow it exactly.

Expected behavior:

1. Read `protocols/evidence-source-maintenance.md`.
2. Read `protocols/provider-access-contract.md`.
3. Discover local-only evidence sources first.
4. Create or update private `EVIDENCE_SOURCES.md`.
5. Record every registry mutation in private `EVIDENCE_SOURCES_CHANGELOG.md`.
6. Do not scan sensitive providers unless they are explicitly enabled and first-scan confirmation is available.
7. Summarize added, updated, skipped, and uncertain sources.
